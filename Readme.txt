GestViz
----------------------------------------------
This package provides the GestViz software.  

This is a class project for the course CS6456 - Principles of UI Software in Fall 2015.

QUICKSTART INSTRUCTIONS :

The prerequisites to run this application are
1. Python 2.8 or above
2. MySQL 
3. Flask API

The system has been tested on MAC and Ubuntu

CODE : 
The size of this software is greater than 5MB. Please clone the code repository from https://github.com/anandsainath/SOREn. 

EXECUTION STEPS:
Start MySQL service
Restore the MySQL dump -gestviz.sql from the data folder

Execute the run.py file

python run.py runserver

URL for the appliction : 
Starting page : http://0.0.0.0:5000/data/main.html
